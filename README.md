# 🚀 Portfolio – strona osobista

Nowoczesna i responsywna strona portfolio stworzona w celu zaprezentowania moich projektów, umiejętności oraz drogi jako Frontend Developer.


## 🌐 Demo

👉 **Live:** *https://portfolio-kacper.netlify.app/*

## ✨ Funkcje

* 🎨 Nowoczesny i responsywny design
* ⚡ Płynne animacje i przejścia
* 🧩 Modularna, komponentowa struktura
* 🚀 Optymalizacja wydajności
* 📂 Sekcja projektów
* 👨‍💻 Sekcja „O mnie”
* 📬 Sekcja kontaktowa
* 🔗 Linki do social mediów

## 🛠️ Technologie

**Frontend:**

* React
* TypeScript
* Vite
* Tailwind CSS
* Framer Motion

**Narzędzia developerskie:**

* Git
* GitHub
* ESLint
* Prettier
